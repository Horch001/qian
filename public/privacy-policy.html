🔒 隐私政策

最后更新日期：2024年11月26日

重要提示：丝绸之路平台（以下简称"我们"或"本平台"）非常重视用户的隐私保护。请仔细阅读本隐私政策，了解我们如何收集、使用、存储和保护您的个人信息。

1. 信息收集

1.1 我们收集的信息

  • Pi Network 账户信息：当您通过 Pi Network 登录时，我们会收集您的用户名、用户ID（UID）和访问令牌
  • 使用数据：浏览记录、搜索历史、交易记录、服务偏好
  • 设备信息：设备类型、操作系统、浏览器类型、IP地址
  • 位置信息：您选择的地区和城市（用于提供本地化服务）

1.2 我们不收集的信息

  • ❌ 我们绝不收集或存储您的 Pi Network 私钥
  • ❌ 我们绝不收集您的助记词或种子短语
  • ❌ 我们绝不要求您提供钱包密码

2. 信息使用

我们使用收集的信息用于以下目的：

  • ✅ 提供和改进平台服务
  • ✅ 处理交易和订单
  • ✅ 个性化用户体验
  • ✅ 发送服务通知和更新
  • ✅ 防止欺诈和滥用行为
  • ✅ 遵守法律法规要求

3. 信息存储与安全

3.1 数据存储

  • 用户信息存储在您的浏览器本地存储（localStorage）中
  • 我们使用加密技术保护敏感数据传输
  • 服务器端数据采用行业标准加密存储

3.2 安全措施

  • 🔐 HTTPS 加密传输
  • 🔐 定期安全审计
  • 🔐 访问控制和权限管理
  • 🔐 数据备份和灾难恢复

4. 信息共享

我们承诺：

  • ❌ 不会出售您的个人信息
  • ❌ 不会向第三方出租您的个人信息
  • ✅ 仅在以下情况下共享信息：
    • 获得您的明确同意
    • 法律法规要求
    • 保护平台和用户的合法权益
    • 与可信赖的服务提供商合作（如支付处理、数据分析）

5. Pi Network 集成

本平台通过 Pi Network 官方 SDK 进行身份验证：

  • 所有登录请求通过 Pi Network 官方服务器处理
  • 我们仅接收 Pi Network 返回的授权信息
  • 您的 Pi 钱包私钥始终由 Pi Network 保管，我们无法访问
  • 支付交易通过 Pi Network 官方支付 API 处理

6. Cookie 和追踪技术

我们使用以下技术改善用户体验：

  • Cookie：记住您的语言偏好和登录状态
  • 本地存储：保存用户设置和缓存数据
  • 分析工具：了解用户如何使用我们的服务

您可以通过浏览器设置管理或禁用 Cookie，但这可能影响部分功能的使用。

7. 您的权利

根据适用的数据保护法律，您拥有以下权利：

  • 📋 访问权：查看我们持有的您的个人信息
  • ✏️ 更正权：更新或修改不准确的信息
  • 🗑️ 删除权：要求删除您的个人信息
  • 🚫 反对权：反对某些数据处理活动
  • 📦 数据可携权：以结构化格式获取您的数据
  • ⏸️ 限制处理权：限制我们处理您的数据

8. 儿童隐私

本平台不面向 13 岁以下儿童。我们不会故意收集 13 岁以下儿童的个人信息。如果您发现我们收集了儿童信息，请立即联系我们，我们将及时删除。

9. 国际数据传输

您的信息可能会被传输到您所在国家/地区以外的服务器。我们会采取适当措施确保您的数据在传输过程中得到充分保护，符合适用的数据保护法律。

10. 隐私政策更新

我们可能会不时更新本隐私政策。重大变更时，我们会通过以下方式通知您：

  • 在平台上发布公告
  • 通过电子邮件通知（如果您提供了邮箱）
  • 在登录时显示更新提示

继续使用我们的服务即表示您接受更新后的隐私政策。

11. 第三方链接

本平台可能包含指向第三方网站的链接。我们不对这些网站的隐私做法负责。建议您在访问第三方网站时查看其隐私政策。

  • 🌐 网站： https://q.toupiao.pro
  • 📱 平台内反馈：通过"个人中心"提交反馈
